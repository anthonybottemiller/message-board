# message-board

#####This project uses ember.js and Google firebase to host a Q and A message board.

#####By Anthony J Bottemiller - 1-16-2017

##Description
This application leverages ember and firebase to build an easy to host message board for Star Trek questions. It could be altered for other subjects easily.

##Technologies Used
* HTML
* CSS
* BOOTSTRAP
* JavaScript
* jQuery
* Node.js
* NPM
* Bower
* browserSync
* Google Firebase

## Requirements

* Modern Web Browser
* [Git](https://git-scm.com/)
* [Node.js](https://nodejs.org/) (with NPM)
* [Bower](https://bower.io/)
* [Ember CLI](https://ember-cli.com/)
* [PhantomJS](http://phantomjs.org/)

## Installation

* `git clone <repository-url>` this repository
* `cd message-board`
* `npm install`
* `bower install`

## Running / Development

* `ember serve`
* Visit your app at [http://localhost:4200](http://localhost:4200).

## Building

* `ember build` (development)
* `ember build --environment production` (production)
